# Listas

Listas desordenadas se indica con guiones
-  item 1
-  item 2
    - item 2.1
    - item 2.2
 

Listas ordenadas se indica con números

1. item 1
2. item 2
   - item 3
  
- [ ] Hacer la cama
- [x] Ducharse
- [ ] Comer
- [ ] Lavarse los dientes
- [ ] Estudiar
